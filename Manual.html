Hotkeys & Funtions

Screen Clipper

Hotkey:
  Tab + LAlt
Function:
  Opening function

Click and drag to set capture area.
If you want to cancel. Click "Rbutton".

It will save image to clip folder.


Hide, Show, and Destroy image

Hotkey:
  Tab + WheelDown
Function:
  Hide image

Hotkey:
  Tab + WheelUp
Function:
  Show image

Hotkey:
  Tab + MButton
Function:
  Destroy image


Open Image (Two types)

Type 1
Hotkey:
  Tab + Space
Function:
  Open Image
Hide, Show, and Destroy with Capslock + "WheelDown, WheelUp, and Mbutton".


Type 2
Hotkey:
  LShift + Space
Function:
  Open Image
Hide, Show, and Destroy with LShift + "WheelDown, WheelUp, and Mbutton".


Copy to clipboard
Notice: If it doesn't work.
Make sure that placing English path and name of folder can not contain space charactors.
When you clip some screen, it will copy the last one to clipboard.


Copy file to opened folder
Hotkey:
  Alt + v
Function:
  Copy Image to opened folder



Other hotkeys

Function:
  Suspend

Hotkey:
  Shift + F11
Function:
  Send clip image to Recycle bin

Hotkey:
  Shift + F12
Function:
  open clip folder
